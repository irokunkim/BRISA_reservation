Supabase 프로젝트 정보를 입력하면 세 기기에서 실시간으로 예약을 공유할 수 있어요.

설정 방법:
1. supabase.com 에서 무료 계정 생성
2. 새 프로젝트 생성
3. Settings → API 에서 URL과 anon key 복사
4. SQL Editor에서 아래 쿼리 실행:

CREATE TABLE reservations ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, date text NOT NULL, time_slot text NOT NULL, platform text NOT NULL, guest_name text NOT NULL, phone text, people integer NOT NULL, memo text, created_at timestamptz DEFAULT now() );
